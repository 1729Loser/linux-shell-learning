# some useful editing commands in vim
    R ==> enters place mode, allowing you to overwrite text continuosly until you press 'Esc'
    J ==> joins the current line with the next one
    d} ==> deletes to the next paragraph
    caw ==> changes a word


# saving and exiting in vim
    :w ==> saves the current state of the file to disk without exiting.
    :q ==> quits the editor, it work's only if you have no unsaved changes.
    :q! ==> quits the editor and discords any unsaved changes.
    :wq ==> first it saves the file and than exit
    :zz ==> it is shortcut and same (equivalent) to :wq
